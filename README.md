## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/FIRE-BCS/finance-pro.git
    ```

2. Navigate to the project directory:
    ```bash
    cd finance-pro
    ```

3. Install the dependencies:
    ```bash
    npm install
    # or
    yarn install
    ```

## Usage

To start the development server, run:
```bash
npm run dev
# or
yarn dev
```
